# Delivering data from a Yandex Managed Service for Apache Kafka® queue to Yandex Managed Service for PostgreSQL through Yandex Data Transfer

With [Data Transfer](https://yandex.cloud/en/docs/data-transfer), you can deliver data from a [Managed Service for Apache Kafka®](https://yandex.cloud/en/docs/managed-kafka) cluster to a [Managed Service for PostgreSQL](https://yandex.cloud/en/docs/managed-postgresql) one. See [this tutorial](https://yandex.cloud/en/docs/data-transfer/tutorials/mkf-to-mpg) to learn how to set up the solution with Terraform. This repository contains the configuration file you will need: [kafka-postgresql.tf](kafka-postgresql.tf).
